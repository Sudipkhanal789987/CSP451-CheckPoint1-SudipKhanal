# CSP451 CheckPoint 1

## Description
This project demonstrates Git, GitHub, version control, branching, pull requests, and basic web development.

Technologies used:
- HTML
- CSS
- JavaScript
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Open the project folder:

```bash
cd csp451-checkpoint1
```

## Usage

Open `index.html` in a browser.

Useful Git commands:

```bash
git status
git log
git branch
```

## Contributing

Create a feature branch before making changes.

Use Conventional Commits:
- feat:
- fix:
- docs:
- style:
- refactor:
